    • Line Official Account Pharmwork
  2. ใส่ข้อมูลในแบบฟอร์มพร้อมลายเซ็นต์ลงนามในเอกสารทั้งหมดให้เรียบร้อย แล้วส่งไฟล์ที่กำหนดดังต่อไปนี้มาทางอีเมล team-support@pharm-work.com เท่านั้น
    1. ไฟล์เอกสารแบบฟอร์ม+สัญญาขอลงโฆษณา
    2. ไฟล์เอกสารแนบประกอบตามที่กำหนดในแบบฟอร์มขอลงโฆษณา
    3. ไฟล์รูปภาพโฆษณาแบนเนอร์ เฉพาะโฆษณาแบนเนอร์ตำแหน่ง A1, A2, A3 และ A4
  3. เมื่อเอกสารได้รับการอนุมัติเรียบร้อย Pharmwork จะส่งลิงค์ชำระเงิน ด้วยระบบ Stripe ผ่านทางอีเมลที่ท่านระบุไว้ในเอกสารแบบฟอร์ม ขอลงโฆษณาเท่านั้น
  4. ชำระเงินผ่านระบบ Stripe ตามลิงค์ที่ทีมงานส่งไปให้ภายใน 5 วันทำการ นับจากวันที่ได้รับลิงค์ชำระเงิน
  5. แนบใบเสร็จชำระเงิน (Receipt) ที่ออกโดย Stripe ส่งมาทางอีเมล team-support@pharm-work.com เท่านั้น
